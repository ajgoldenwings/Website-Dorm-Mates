Positioning

The position property specifies the type of positioning method used for an element.

  • static - positioned normally as if it was a div element, top right bottom left properties do not apply
  • relative - positioned normally however top right bottom left properties do apply which act similarly like margin
  • fixed - positioned to viewport, the most parent, stays in the same place even when scroling
  • absolute - positioned to closest parent that has a position property

The position property must be set first before using top, bottom, left, and right properties.
